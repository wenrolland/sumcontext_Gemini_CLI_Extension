# sumcontext

## Aperçu

Après de longues minutes, ou même heures, de conversation avec Gemini, il est probable que vous ayez besoin de faire le point sur tout ce qui a été dit. La commande `/sumcontext` est une commande personnalisée qui permet de générer un résumé structuré de la conversation.

### Comment l'utiliser

Rien de plus simple !

- Après une conversation bien remplie d'informations, entrez la commande `/sumcontext`.
- Le LLM générera un sommaire du contexte depuis le début de la conversation ou jusqu'au plus récent sommaire.

## Installation pour Gemini CLI

Installer directement depuis GitHub :

```bash
gemini extensions install https://github.com/wenrolland/sumcontext
```
